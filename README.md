Python Snake foi um projeto acadêmico no qual desenvolvi uma releitura moderna do clássico jogo Snake, utilizando Python com a biblioteca Pygame. 
A aplicação conta com ajuste de dificuldade, sistema de pontuação com registro de recorde e persistência de dados. 
O jogo também inclui efeitos sonoros para movimentação e alimentação, além de música de fundo em loop, tornando a jogabilidade mais imersiva.
A arquitetura foi estruturada de forma modular, com separação em classes responsáveis por nível, cobra, comida, pontuação e menu,  garantindo 
melhor organização, manutenção e escalabilidade do código. 
O resultado é uma aplicação com jogabilidade fluida, visual simples e código bem estruturado, demonstrando domínio de Python, lógica de 
programação, desenvolvimento de jogos e utilização de bibliotecas gráficas.
O jogo  se encontra disponivel para Windows e Linux
